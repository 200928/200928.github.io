数据宽度

C C++ Java 都需要定义数据的类型。计算机底层需要我们给这些数据定义宽度

位 0 1
字节 0 - 0XFF
字 0 - 0XFFFF
双字 0 - 0XFFFFFFFF

有符号数和无符号数

数据都是有宽度的，每个数据代表什么意思？

规则

二进制解码增加一个规则

  • 无符号数规则

  • 有符号数规则

    最高位是符号位：1（负数）0（正数）

原码反码补码

编码规则 ：有符号位的编码规则

  • 原码：最高位是符号位
  • 反码：
    • 正数：反码和原码相同
    • 负数：符号位一定是1，其余对原码取反
  • 补码：
    • 正数：补码和原码相同
    • 负数：符号位一定是1，反码 +1

例子

一个二进制数，需要了解它是有符号数还是无符号数

如一个 32 位机器

FFFF FFFF

  • 有符号表示 -1
  • 无符号表示 4,294,967,295

  3. 异或运算（xor ^）不同为1

  4. 非运算（单目运算符 not ~）

    0 就是 1，1 就是 0

  5. 位运算（移动位，左移 * 2，右移 / 2）

    左移：shl <<

    右移：shr >>
