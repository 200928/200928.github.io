快捷键

  1. 创建内容：Alt + Ins
  2. main() ：psvm
  3. 复制行：Ctrl + D
  4. 删除行：Ctrl + Y
  5. 代码上/下 移动：Ctrl + Shift + Up/Dn
  6. 收缩类：Ctrl + N
  7. 生成代码：Alt + Ins (如构造器, hashcode)
  8. 百能快捷键：Alt + Enter （导包，生成变量等）
  9. 单行注释或者多行注释：Ctrl + / 、Ctrl + Shift + /
  10. 重命名：Shift + F6
  11. for循环：fori
  12. 代码块包围：Ctrl + Alt + T
  13. 自动补全提示：Alt + /
  14. 代码一层层调用：
    点进源码：Ctrl + 鼠标悬浮 + 点进去
  15. 左右回退：
  16. 显示代码结构：Alt + 7
  17. 显示导航：Alt + 1
