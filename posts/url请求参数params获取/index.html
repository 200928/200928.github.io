URL请求参数Params获取

xiaozao included in 学习
 2023-10-18  370 words   One minute   views
Contents
  • 参考

获取当前路径的 url 可以使用Window：location 属性，然后通过这个 url 创建一个 JavaScript 内置 URL 类的实例对象

1
2

const url = new URL(window.location)
const param = url.searchParams.get("{{param}}")

这样看似乎没有问题，先获取 URL 对象，然后用searchParams属性得到请求参数，最后用get方法获取请求参数

searchParams属性：返回一个URLSearchParams对象，可以调用URLSearchParams对象各种方法，对查询字符串进行非常方便的处理

下面看个例子，这里例子没有任何问题，可以获取 url 中的 params

 1
 2
 3
 4
 5

 7

const url = new URL("https://example.com/a/b?foo=1&bar=2")
const foo = url.searchParams.get("foo")
const bar = url.searchParams.get("bar")

console.log("foo", foo)
console.log("bar", bar)

// 输出结果
foo 1
bar 2

const params = new URLSearchParams("https://example.com/a/#/b?foo=1&bar=2")

const foo = params.get("foo")
const bar = params.get("bar")

console.log("foo", foo)
console.log("bar", bar)
